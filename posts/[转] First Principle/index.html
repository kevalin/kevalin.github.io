[转] First Principle

本文首发 GURU Magazine； 作者：MR.SOPHIE， 欢迎转载，但请自觉保留以上版权声明。

「我会运用「第一性原理」思维而不是「类比」思维去思考问题。在日常生活中，人总是倾向于比较 — — 别人已经做过了或者正在做这件事情，我们也就去做。这样的结果只能产生细小的叠代发展。 「第一性原理」的思考方式是用物理学的角度看待世界的方法，也就是说一层层剥开事物的表象，看到里面的本质，然后再从本质一层层往上走。 」 — SpaceX、Tesla 电动汽车 及 PayPal 创办人 Elon Musk

什么是 「第一性原理」（ First Principle )？

所谓的「第一性原理」是一个量子力学中的一个术语，意思是从头开始计算，只采用最基本的事实，然后根据事实推论，创造出新价值。在 Elon Musk 开发 Tesla 特斯拉电动车案例中，很多专家觉得电动车是不可能流行起来，因为电池成本在历史上一直也降不下来。 600美元 / 千瓦是市场的公价，电池从一直也是那么贵，它的改进和降价总是很慢，所以它未来短时间内也不大可能大幅度降低价格。

但Elon Musk 却不认同，在他公司新电池的开发阶段中，他率先屏弃现时市场所有生产电池组的已有技术，把电池组的构成物质全部分解，还原成最基础的材料：碳、镍、铝及其他用于分离的聚合物，这种还原使他了解到重新构成了制造电池的**「基本事实」（ Fact ）**是什么。

无可否认，上述的金属成本如果在市场需求没有大幅度改变下，是绝对降不下去的，可是他却发现了当中剩下来的成本还包含了很大部份是属于「人类协作过程」而生的成本，而他相信凡是人类协进的事情，就必定存在优化空间。

透过这些「基本事实」，Elon Musk 和团队再把原材料每个部分再细致分析及实验，并把每项工作流程再优化重组，比如，在美国生产可能税费比较高，那就不要在美国生产了；某种原有技术的模块设计上出了问题，那就改变设计，最后他和团队把各部份优化原件，加上全面改良的生产方法，整合成现时以能大幅度降低电池的生产成本为前提的电动汽车。

而把「第一性原理」的思想放在Elon Musk 的SpaceX 计划，他也同样挑战过去太空运输技输产业中「成本就是那么贵」的专家偏见，他先还原制造火箭「基本事实」，发现了一架火箭的原料成本原来只占火箭的总成本的2%，而余下的成本其实是其他制造过程的成本，而有了这层认知，他便朝着优化另外98% 的成本方向，把现时制造火箭的成本，降低了到现时的10% 。

这就是「第一性原理」（ First Principle ) 的爆炸力。

可是为什么我们明明和 Elon Musk 身处在同一个世界，却看不到 Elon Musk 看到的「第一性原理」（ First Principle )？为什么？难道真的只是因为他比较有钱，接触到较多高级知识份子吗？总结原因，我认为有三大理由:

一、我们看不到，因为我们缺乏「硬学科」训练

「第一性原理」（ First Principle ) 其实是事物底层规律的总结，就以泥石流作为例子，当你知道「从山顶上滚下的石头会愈来愈快」这个基本事实后，如果当你不幸遇上泥石流时，你会选择尽可能往山的两侧跑，而不是和顺着山谷和泥石比拼斗快，这个知识对你来说，可算是「野外求生」的知识，然而如果你能把这个知识发掘到底层，它其实就是为牛顿第二定律F＝ma，有了这个底层知识，你不单能避开泥石流，更有可能想出造火箭方法。

而你能把这个大家也看得见的眼前「基本事实」，或「野外求生」知识，向底层发掘为大家也无法轻易以肉见看见的牛顿第二定律F＝ma，需要的就是「硬学科」，例如数学、物理及化学。这些「硬学科」也许我们在求学时期早已学过，但在现在日常生活中，或许只余下发薪水或买菜时，常用的加减乘除外，已无用武之地。

那为什么我们从不会思考过如何融会贯通地使用呢？因为我们不明白这些「硬学科」价值在哪里。

相比起心理学、经济学和社会学等人文学科需经常配搭前置假设才能应用，「硬学科」是完全建立在基础假设及逻辑思维分析之上，例如数学就是一个完全不依托真实存在的世界，透过假定范围，几乎所有的推论都是正确，因此它的知识可以算是更可靠，更贴近「第一性原理」（ First Principle ) 的本质。

二、我们看不到，因我们「自以为知道」

在一般学习书借经常提到 : 个人认知的「知道」与事实上的「知道」的四象限中，我尝试提再把它演绎为四个不同的层次 :

  • 「不知道自己不知道」Level 1 ：以为自己什么都知道，自以为是的认知状态
  • 「知道自己不知道」Level 2 ：有敬畏之心，开始空杯心态，准备好投入学习
  • 「知道自己知道」Level 3 ：抓住了事情的规律，提升了自己的认知
  • 「极致的意会」Level 4：对事情的掌握，已经变成一种浑然天成的意会，在别人辗转思量之际，你已立即能下准确的决定

「认知」几乎是人和人之间唯一的本质差别，技能的差别是可量化，但认知的差别却是本质性的，不可量化。人和人比拼的除了是实践力外，更重要是洞察力。

你的求知欲通常是由「你知道了自己不知道」（Level 2 ）开始产生; 人选择不去求知，主要是因为大部份人一直也停留在「不知道自己不知道」（Level 1 ） 。

「不知道自己不知道」（Level 1 ）的状态是因为自己连那个「不知道」是什么都没有搞清楚，这就好比西医只知「发炎」，而不知何谓「上火」。

对中医来说，西医所谓的「发炎」（ Inflammation ），其实是指「上火」，而火是有「实火」与「虚火」之分，而在虚实之中，治疗方法也是可以完全截然相反。

而西医却因为从不知「上火」一字（ 或可以说就算就知道，也不重视「上火」在西方医学知识系统的融合），只相信「发炎」便能解释一切现象，因此亦错过了在辨症时，以虚实之火去下更准确的药方的机会，也错过了自己发掘应对炎症不同程度症状的新启发，这就是「不知道自己不知道」（Level 1 ）的状态所引发的问题。

三、我们看不到，因为我们「急功近利」的学习态度

学习是需要「基本功」的累积，凡事追根究底，深入学习，是要经历流汗、未知、脑汁和时间付出。在华人以「考试结果及职业导向为最终学习目的」的情况下，我们早已失去了对学习的深索热情和乐趣。

当你身边人也在职场的高速公路上怒奔，大家终日也在看「三分钟学会Google 的创新法则」，「三十分钟不败精读法」，「三天快速增加你的财富收入」，并和你吹嘘着上述的方法是如何启发及有效，在创业场或职场上同样具有竞争心的你怎能不焦急？在这里我和你谈学习需要时间练「基本功」，你也许会想 : 「别人都已进步都那么快，再谈基本功我就已做大输家了！」

可是请停一停，让我们能否用科学化的方法，再重新思考一下:

在正常人的能力成长曲线中，其曲线的前期一般会随着学会了具体方法和技术后快速增加，我们解决问题的时间会愈来愈短，对一些开始时还是有难度的事情，到达中期顶峰阶段，经过练习后就会变得易如反掌，可是这个成长曲线到达后期就会失去向上升的动力，为什么？

因为我们大多数人在日常认识问题时，一般只会依靠直觉、个人经验、简单的线性思维、因果关系、意识形态和价值观偏好，而这些思维却会引发 :

  1. 我们无法发现事情之间深层次的关联，我们眼前的认知都是一个分散的点，是一种孤立且断裂式的认知，例如你无法明白到底SpaceX 和 Tesla 电动汽车到底有什么关系？
  2. 我们面对超出自己日常工作的问题时，不知从何下手，更无法准确把握关键环节并合埋地预测事情的发展趋势，例如你无法理解如何由电池组的构成基本原素，预测到解决澳洲电力危机的解决方向？

我们经常都听到身边那些在职场闯荡了几年的人会埋怨自己在公司已学不到任何新事物，感觉成长已到达天花板，真正原因不是你成长得太快，而是因为你的天花板太矮了。这个天花板，就是由你急功近利的学习方法所造成，因为你只看到天花板一个个孤立的点，而看不到天花板外原来还有楼宇的钢筋水泥结构，城市空间的规划原则，城市的发展的建筑历史。

相反如果我们能反其道，以慢打快，采用「第一性原理」（ First Principle ) 的学习原则，我们的成长曲线就会出现这个模样 :

我们在学习的前期，虽然会因自己需不断训练和掌握基本原则，而令学习速度变慢，但当我们掌握了整个学科的理念和方法后，学习的能力就会大幅提升。

你可以透过「第一性原理」（ First Principle ) ，从底层的规律，以跨领域的方式，不停地活泼游走并累积，而随着你的知识愈多，你的成长曲线会增长得愈来愈快，而当你能整合的知识愈多，你的知识就开始产生了爆炸性的威力（股神巴菲特最亲密的战友Charlie Munger 称之为「Lollapalooza Effect」），透过这种学习和成长，你会更容易获得对未来更准确的「预测」，从而获得先机，成为产业中的新先知。

钻研知识的路，从不挤拥

我曾经听过长辈感叹: 「今天是一个资讯和知识爆炸的社会，比起以往互联网年代前的世界，当年的世界单纯和清静好多。」我认为这个观念是谬误，人类文明的发展，本来就是包含着混乱和喧闹，以往的世界你觉得清静，是因为讯息传递缺乏效率，而讯息内容的力量在传递的过程中，也会像热力传递过程中会逐步递减，所以接受者才不会有现在如直播般的「冲击」。

同时，我们必须在一片「资讯和知识已爆炸」喧闹声，重新分清在这些爆炸中，到底什么是「资料」、「资讯」和「知识」（这个分类将会在下一篇文章详细解释），现今的社会爆炸的是「资讯」，更正确来说是「垃圾资讯」，而非知识。知识的制造门槛是极高，并非你说爆就爆，因此钻研知识的路，是又阔又人烟稀少，你以为人多的部份其实也不过追求快速「学习具体技巧」的方法论人群，它们和我们今天所分享的「第一性原理」（ First Principle ) 或底层定律，是完全在处于不同的程度。

总结今天的分享内容，我们理解了 :

  1. 「第一性原理」（ First Principle ) 的定义
  2. 我们看不到的「第一性原理」（ First Principle ) 的原因 : 缺乏「硬学科」训练、「自以为知道」「急功近利」的学习态度
  3. 我们学习「第一性原理」（ First Principle ) 的好处 : 获得长远累进的成长曲线; 得到对未来的洞见及获得机遇

由今天起，让我们一起刻意练习（ Deliberate Practice ) ：

  1. 回想最近几年，有什么知识是你当初认为是不重要，但后来你才后悔自己没有早点知道？
  2. 反思自己在上述过程中，有什么关键的事件、人物或原因令你醒觉上述的知识真的很重要？
  3. 尝试运用「第一性原理」（ First Principle ) 的思考方式，发掘出你在学习认知中，那些经常见到但自己却一直没有观察到事情，并找出改良方法，例如 :
  • 为什么我对数字总是很不敏感？原来过去我总会以「人类是有血有肉，不能被量化」和「人的灵感直觉比机械式操作更重要」这类借口，轻忽了逃避学习数理 ;
  • 那为什么我会轻忽数理的重要性？因为我是人文学科的人，所以每次面对数理相关的问题都会总是很没有安全感，觉得自己比理科生低人一等……
